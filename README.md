# Little Mage

A fantasy action-adventure game built in **Unreal Engine 5**, combining retro pixel-art visuals with modern 3D gameplay systems.

---

## Overview

Little Mage is a small-scale RPG-style game where players control a wizard exploring magical environments, casting spells, and battling enemies. The project focuses on blending **2D pixel art** with **UE5’s 3D engine**, supported by original music and sound design.

---

## Built With

- **Unreal Engine 5** – Game engine
- **Aseprite** – Pixel art sprites and UI assets
- **FL Studio** – Soundtrack and sound effects

---

## Installation / How to Run

1. Clone or download the repository
2. Open the project in **Unreal Engine 5.3 or later**
3. Load the main level
4. Press **Play** in the editor

---

## Controls

- **WASD** – Move
- **Left Click** – Spell 1
- **Right Click** – Spell 2
- **ESC/P** – Pause / Menu

---

## Inspiration

Little Mage was inspired by classic fantasy RPGs and indie games that use charming visuals and immersive audio. The goal was to create a nostalgic experience using pixel art while leveraging modern tools for gameplay and presentation.
